Contrast

Github

github.com

This Github dashboard page illustrates the principle of contrast in design very well. The background is a very dark colour and the contents are grouped in a lighter shade of gray, such that it is easy to see which elements on the page are related. The green highlighting also makes it easy to see, at a glance, which areaas of the page to interact with.

Proximity

Jumia

www.jumia.com.ng/phones-tablets/

On this e-commerce site, the different items are placed close to each other so that there is a constant stream of merchandise to look at. This improves user engagement with the site, because it makes the user more likely to view many item. The site relies on volume, so if the items were too spaced out, a user could get bored or uninterested before the next item finally shows up.

Whitespace

Google

google.com

The Google homepage is probably one of the most minimalist out there. It is uses whitespace to focus your attention on the search bar and search suggestions. This reinforces that Google.com is a search engine.
